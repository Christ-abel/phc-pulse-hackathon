# openIMIS Frontend PHC Pulse module

React module providing the UI for capitation/OBA payment management, built
for Track 2 of the openIMIS Hackathon.

## Status

Scaffold only (Phase 1): registers a placeholder menu entry and route into
the openIMIS navigation. Real pages (Facility Dashboard, Scheme Console,
FHIR PaymentNotice viewer) are added in Phase 5.

## Installation

Installed like any other openIMIS frontend module: add an entry to the core
shell's `openimis.json` pointing at this package, then rebuild the frontend.
